# Tailwind SaaS Dashboard Front-End

A modern SaaS dashboard built with Next.js 14, React 18, and Tailwind CSS.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Set up the database:
```bash
npx prisma db push
npx prisma db seed
```

3. Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Dashboard with analytics
- User management
- Billing integration
- Reports and data visualization
- Authentication with NextAuth.js
- Responsive design

## Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Prisma ORM
- NextAuth.js
- Recharts
